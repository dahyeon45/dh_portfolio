1. Personal photo & music archive

▷ URL
https://dahyeon45.github.io/dhrchive/

▷ Details
1) Spotify Web API
본 프로젝트의 첫 번째 목표는 Spotify Web API를 활용해보는 것이었습니다. Web API의 기능을 프로젝트에 적용하기 위해 authorization부터 요청을 보내고 응답을 처리하는 것까지 전 과정을 공부하였습니다. 해당 과정을 정리한 글은 다음의 링크(Link)🔗에서 확인하실 수 있습니다.
2) React
리액트를 활용해 제작함으로써 리액트의 라우팅, 훅에 좀 더 익숙해질 수 있었습니다.
3) CSS
기존의 블로그보다 인터랙티브한 사이트를 제작하고자 노력했고, CSS animation을 적용하는 등 CSS styling에 공을 들였습니다. 세부 페이지는 아직 미완성이나, 사진에 따라 설정된 음악이 재생될 수 있도록 제작할 예정입니다.

2. Audio Reactive Visuals
▷ About
음악에 반응하여 별이 반짝이는 아트워크입니다. p5.js를 이용하여 제작하였습니다.

▷ URL
https://dahyeon45.github.io/audioreactivePL/

▷ Details
소리의 진동수 구간을 128개로 나누고 128개의 별을 캔버스에 랜덤하게 찍어, 한 개의 별이 한 개의 구간을 대표하도록 하였습니다. p5.sound 라이브러리의 FFT를 통해 각 진동수 구간의 진폭을 담은 배열을 얻었고, 해당 구간의 진폭을 별의 크기로 변환함으로써 진폭에 따라 별의 크기가 달라지게 하였습니다.
